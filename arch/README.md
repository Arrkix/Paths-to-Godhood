<h2>Background</h2>
For anyone contemplating to install arch themselves. These are a few tips I personally think could be helpful. 
	-Basic linux commands: Navigation, reading errors, should be second nature. [ I had experience using ubuntu in vm for a few weeks.]
	-Linux system: Good understanding of lower level components of OS, such as partitions, filesystems, mounting. [Nothing in-depth, I read few chapters of linux basic and knew these concepts.]
	-OS: Concepts of booting how hardware and softwares interact. [Again just the overall ideas.]
	-Linux installing: Brief through the basic workflow of installing an OS. [Watch videos, or read articles of the steps of installing Arch.]

	=><u><b>The Arch Installation Guide<b>: The official Arch installing documentation. To be able to read through it, follow its instruction, rereference as many times as required. <u>
		
		This can't be overstressed, of how important the offical Documentation is. It is robust, exhaustive and complete. I made the mistake of following multiple guides/ videos and ended up scarring away several hours of the install. Developing the skill to push through pages of documentation is an invaluable skill, and for arch it is a life saving path that can save hours of headache adn worse.
